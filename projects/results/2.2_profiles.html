Profiles per Benchmark

Rankings from the results of the 26 measured compilation profiles considering each of the 12 benchmarks individually. For each case are indicated the classification according to the execution time in seconds, CPU and memory RAM energy consumption (individually and together) in Joules and to the ratio between both of these factors (energy consumption/time) in Joules/seconds. The classifications represent the ranking of the mean value obtained from the set of each program executions, after being compiled with the respective profile. For each classification were created four versions that differ in the number of decimal places considered (0, 1, 2 or 3). In the auxiliary files Proj2AuxInfo and Proj2AuxResults are provided some more results as well as described other relevant information, namely to which elements corresponds each of the presented IDs.
