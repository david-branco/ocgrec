Tools per Benchmark

Rankings from the results of the 18 software development tools considering each of the 12 benchmarks individually. For each case are indicated the classifications according to the execution time in seconds, CPU and memory RAM energy consumption (individually and together) in Joules and to the ratio between both of these factors (energy consumption/time) in Joules/seconds. The ranking of each tool represent a mean value of their compilation profiles, according to the mean values obtained from the set of each program executions after being compiled with the respective profile. Each element has in addition to its overall classification for a particular measure, the sum of the various classifications that it obtained individually for the various cases. For example, if a tool has only the best 2 profiles for a given strand, then in this instance it will have the rank 1 and also the respective sum (2). For each classification were created four versions that differ in the number of decimal places considered (0, 1, 2 or 3). In the auxiliary files Proj2AuxInfo and Proj2AuxResults are provided some more results as well as described other relevant information, namely to which options are contained in each compilation profile.
