Brief Description

Result set for the 12 programs analyzed using 6 default compiler optimization levels. For each case are indicated the obtained values of the execution time (in seconds) and the respective energy consumptions of CPU, RAM and GPU (in Joules). They represent the mean value obtained from the set of each program executions, after being compiled with the respective optimization level. In the auxiliary file Proj1AuxInfo are provided some more results as well as described other relevant information.
