Tools Default Profiles

Joint results of the 18 software development tools according to their default compilation profile for each of the 12 benchmarks under study. For each case are indicated the obtained values of the execution time (in seconds) and the respective energy consumptions of CPU and RAM (in Joules). They represent the mean value obtained from the set of each program executions, after being compiled with the respective tool default profile. In the auxiliary files Proj2AuxInfo and Proj2AuxResults are provided some more results as well as described other relevant information, namely to which elements corresponds each of the presented IDs.
