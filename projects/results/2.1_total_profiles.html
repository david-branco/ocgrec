Profiles per Benchmarks

Rankings from the results of the 26 measured compilation profiles considering all the 12 benchmarks jointly. For each case are indicated the classifications according to the execution time in seconds, CPU and memory RAM energy consumption (individually and together) in Joules and to the ratio between both of these factors (energy consumption/time) in Joules/seconds. They represent a global ranking that considers the classifications of the mean values obtained from the set of each program executions, after being compiled with the respective profile. Each element has in addition to its overall classification for a particular measure, the sum of the various classifications that it obtained individually for the various cases. For example, if a profile is the best for all the benchmarks for a given strand, then in this instance it will have the rank 1 and also the respective sum (12). For each classification were created four versions that differ in the number of decimal places considered (0, 1, 2 or 3). In the auxiliary files Proj2AuxInfo and Proj2AuxResults are provided some more results as well as described other relevant information, namely to which elements corresponds each of the presented IDs.
